

# Task 6 - Sales Trend Analysis Using Aggregations

- GitHub

- ORDER BY
- LIMIT
- WHERE

## Analysis Performed

2. Monthly Order Volume Analysis

4. Top 3 Months by Revenue
5. Time Period Analysis
6. Average Monthly Revenue
7. Highest Revenue Month

Successfully analyzed sales trends and identified high-performing months using SQL aggregations.
